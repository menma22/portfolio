# New Fashion - AIバーチャル試着アプリ

## 1. プロダクト概要
* **コンセプト**
  「買う前に、着る。ファッションECの新しいスタンダード」
* **概要**
  生成AI（Gemini API）を活用し、自分の写真に気になった服を自然に合成できるバーチャル試着アプリケーション。試着から購入、コーデ共有までをワンストップで提供します。

## 2. 開発のきっかけ・背景
**「ネット通販の『思ってたのと違う』をなくしたい」**

私自身ファッションが大好きですが、ECサイトでの購入には常に「サイズ感や雰囲気が自分に合うか分からない」というリスクが伴います。「買ってみたけど着なかった」という失敗や、返品の手間を減らしたいと強く思っていました。

そんな中、**Gemini Nano / Pro**（生成AI）の画像生成能力に触れ、「これを応用すれば、実際に服を着なくても、まるで着ているかのような自然な試着画像が作れるのではないか？」と確信し、開発に着手しました。

## 3. 主な機能と体験
* **AIバーチャル試着（Virtual Try-On）**
  ユーザーの全身写真と、ECサイト上の服の画像をAIに読み込ませることで、瞬時に着用画像を生成。「自分の手持ちの服に、新しい服が合うか」を視覚的に確認できます。
* **バーチャル・クローゼット**
  気になった服や生成したコーデ画像をSupabase上に保存。購入前の検討リストとしてだけでなく、手持ち服の管理ツールとしても機能します。
* **コーデシェアSNS（構想・一部実装）**
  作成したバーチャルコーデをInstagramのようにシェアできる機能。「服を買うお金がなくても、センスだけで勝負できる」世界観を目指し、インフルエンサーがバーチャル上でスタイリングを提案できるプラットフォームを構想しました。

## 4. 技術的な課題と考察
**「見た目」と「物理的なサイズ感」の壁**

開発を通して直面した最大の課題は、**「生成AIは『着ている風』の画像を作るのは得意だが、厳密な『サイズ感（着丈・袖丈）』の再現は苦手である」**という点です。

AIはピクセル単位の合成は行いますが、物理シミュレーションを行っているわけではないため、「SサイズとMサイズの違い」などを正確に描写することに限界を感じました。
これは現在のファッションテック業界全体の課題でもありますが、今後は3Dモデリング技術との併用などで解決策を探っていきたいと考えています。
